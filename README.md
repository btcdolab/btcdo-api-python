##### 1.登录Btcdo账户，在“个人中心”→“安全中心”获取API-KEY和API-SECRET

##### 2.搭建python3.6环境

###### 以Mac OS X系统为例，下载安装包：

```
https://www.python.org/ftp/python/3.6.4/python-3.6.4-macosx10.6.pkg
```

###### 安装pip3

```shell
pip3 install numpy 
```

###### 安装certifi

```shell
/Applications/Python\ 3.6/Install\ Certificates.command
```

##### 3.执行rest-api.py查看结果

```shell
python3 rest-api.py 
```
